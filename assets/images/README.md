# Assets / Images

Letakkan file logo toko di sini dengan nama:

```
logo.png
```

Logo akan ditampilkan otomatis di bagian atas halaman home (`users/pages/home.php`).

Tinggi tampilan default: 96px (mobile) / 128px (desktop). Disarankan pakai PNG transparan dengan ukuran minimal 256x256 px supaya tetap tajam di layar HiDPI.

Jika file `logo.png` belum ada, gambar otomatis disembunyikan (`onerror` handler) — judul "Hadiah Spesial" tetap tampil normal.
